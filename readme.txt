Reduce image file sizes and improve performance using the <a href="http://dietimage.com/">dietimage.com</a> API within WordPress.

== Description ==

dietIMAGE uses smart lossy compression techniques to reduce the file size of your PNG files. By selectively decreasing the number of colours in the image, fewer bytes are required to store the data. The effect is nearly invisible but it makes a very large difference in file size!

Features :

Image Compression
- Easy to compressed your media image
- Bulk compression
-Auto compression of your media image
-Restore original image
-ON/Off auto compression

HTML/CSS and Javascript Compression
-Compressed HTML/CSS and javascript
-On/Off HTML/CSS and javascript compressed

WP THEME CSS
-Compressed style.css of your theme

== Installation ==

You can find, download and install WP Easy Tools Compression directly from the **Plugins** section in WordPress.
or go to this link http://codemywebapps.com/wp-easy-tools-compression/

If you want to install manually, download and unzip the wp-easy-tools.zip file and upload to the `/wp-content/plugins` directory. Then activate the plugin through the **Plugins** section in WordPress.
